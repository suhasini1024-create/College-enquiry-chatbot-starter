# College Enquiry Chatbot

A simple rule-based College Enquiry Chatbot developed using Java.  
This chatbot assists students by providing instant responses to common 
college-related queries such as admissions, courses, fees, timings, 
placements, and contact information.

---

## Project Features

- Command Line Interface (CLI) based chatbot
- Rule-based keyword matching
- Modular and maintainable code structure
- User-friendly `help` command
- Graceful handling of unknown queries
