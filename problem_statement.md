## Problem Statement

Students often face difficulty in getting quick and accurate information 
about college-related queries such as admissions, courses, fees, timings, 
placements, and contact details.

Manual enquiry systems are time-consuming and require human assistance. 
Hence, there is a need for an automated chatbot system that can respond 
to common college-related questions instantly.

This project aims to develop a College Enquiry Chatbot using Java that 
provides accurate information to students through a simple interactive interface.
